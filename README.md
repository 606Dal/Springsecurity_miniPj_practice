## 스프링 시큐리티 – 인증 관리 (작은 웹 애플리케이션)
---
#### 개발 환경

1. 운영체제 : windows10
2. STS4 (spring-boot)
3. JavaSE-17
4. 데이터베이스 : MySQL 8

---
기간 : 2024-06-21 ~ 22
* 양식 로그인 인증 방법 이용, 사용자 세부 정보는 데이터베이스에 저장, 맞춤형 인증 논리 구현
(WebSecurityConfigurerAdapter 대신 SecurityFilterChain으로 바꾸는 부분이 조금 오래 걸림)
